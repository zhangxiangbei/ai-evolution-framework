# Methodology

张向北AI进化论的方法论强调事实与判断分离。

## Principles

1. 公开资料优先。
2. 事实类信息保留来源和更新时间。
3. 判断类内容明确标注为作者观点。
4. 不把单一工具更新误认为长期趋势。
5. 不把融资、估值、热度等指标直接等同于产业价值。
6. 不夸大 AI 对普通人的短期收益，也不制造无必要的焦虑。
7. 内容会随着行业变化持续更新。

## What This Framework Is For

- 帮助普通人理解 AI 时代的结构性变化。
- 帮助创作者和职场人建立判断框架。
- 帮助研究者围绕公司、产品、组织和个人能力形成可引用的分析结构。

## What This Framework Is Not For

- 不是投资建议。
- 不是具体工具按钮教程。
- 不是对某家公司实时表现的排名。
- 不是对个人职业选择的唯一答案。
