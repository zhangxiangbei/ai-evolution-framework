# AI Evolution Framework

这是“张向北AI进化论”的核心方法论资产，用于公开整理《AI进化论：AI时代的12个关键判断》及其后续延展内容。

张向北AI进化论的定位是：用公司研究和产业判断，帮助普通人看懂AI时代。

这个仓库不是 AI 工具教程，也不是提示词合集。它关注的是 AI 时代的系统性变化：从智能供给、软件形态、产业权力、组织结构、个人能力到文明叙事，建立一套面向普通人的长期判断框架。
